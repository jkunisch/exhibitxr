## 2. Browser Agent Prompt (für Comet / Claude Computer Use)
Kopiere diesen Prompt und gib ihn deinem Browser-Agenten, sobald die Domain live geschaltet ist:

> **System-Prompt für Browser Agent:**
> "Navigiere zur Google Search Console unter https://search.google.com/search-console.
>
> 1. **Property auswählen:** Suche nach der Property 'https://3d-snap.com'. Falls sie nicht existiert, lege eine neue 'URL-Präfix' Property für diese Adresse an.
> 2. **Sitemaps aufrufen:** Navigiere im linken Menü zum Punkt 'Sitemaps' (unter 'Indexierung').
> 3. **Neue Sitemap hinzufügen:** Gib in das Feld 'Neue Sitemap hinzufügen' den Pfad `sitemap.xml` ein.
> 4. **Absenden:** Klicke auf 'Senden' und bestätige den Erfolg.
> 5. **Indexierung anstoßen:** Gehe zur 'URL-Prüfung', gib `https://3d-snap.com/` ein und klicke auf 'Indexierung beantragen', um Google einen sofortigen Ping zu geben.
>
> Melde Vollzug, sobald die Sitemap den Status 'Erfolgreich' anzeigt."

---

## 3. Nächste Schritte nach dem Deployment
1. **DNS Check:** Stelle sicher, dass `3d-snap.com` erreichbar ist.
2. **Sitemap Validierung:** Rufe `https://3d-snap.com/sitemap.xml` manuell auf, um sicherzustellen, dass keine 404 Fehler vorliegen.
3. **Prompt ausführen:** Browser-Agent starten.
